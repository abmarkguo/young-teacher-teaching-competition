# 竞赛层级要求矩阵

> 三级竞赛（校赛/省赛/国赛）标准对比矩阵

---

## 一、基本要求对比

| 维度 | 校赛 | 省赛 | 国赛 |
|------|------|------|------|
| **教学设计数量** | 2-3个学时 | 5个学时 | 16个学时 |
| **PPT数量** | 2-3套 | 5套 | 16套 |
| **课堂教学时长** | 15分钟 | 20分钟 | 20分钟 |
| **教学反思** | 500字/45分钟 | 500字/45分钟 | 500字/45分钟 |
| **抽签方式** | 自选或抽签 | 抽签 | 现场随机抽签 |
| **赛前通知** | 提前通知 | 提前通知 | 赛前1小时告知 |
| **直播** | 无 | 无 | 央广网全程直播 |
| **参赛人数** | 各院推荐 | 各省选拔 | 161名（第七届） |

## 四、格式要求（统一标准）

### 教学设计格式
- 一级标题：3号黑体加粗
- 二级标题：4号黑体加粗
- 三级标题：小4号黑体加粗
- 正文内容：小4号宋体，1.5倍行距
- 格式为PDF文档通用格式

### PPT格式
- PowerPoint演示文稿
- 16:9比例
- 与教学设计内容对应一致

### 封面格式
- 第一行：竞赛材料名称（页面左上角，二号，宋体）
- 第二行：课程名称（页面三分之一处居中，初号，宋体）
- 第三行：学院及姓名（页面二分之一处居中，一号，宋体）

### 禁止事项
- 教学材料不得出现姓名、学院(系)等任何个人相关信息
- 教学反思不允许携带任何书面或电子等形式的资料

## 五、评分权重（统一标准）

| 环节 | 权重 | 满分 |
|------|------|------|
| 教学设计 | 20% | 20分 |
| 课堂教学 | 75% | 75分 |
| 教学反思 | 5% | 5分 |
| **总计** | **100%** | **100分** |

> 三级竞赛评分权重相同，差异在评审严格度和材料数量要求。

## 六、晋级路径

```
校赛（各院/校组织）
  ↓ 各组别推荐
省赛（各省教育工会组织）
  ↓ 各组别推荐1名+协办省份1名
国赛（中华全国总工会+教育部组织）
  ↓ 现场决赛
获奖（一/二/三等奖）
```

### 晋级要求
- **校赛→省赛**：各组别一等奖前三名
- **省赛→国赛**：各省各组别推荐1名（协办省份+1名）
- **国赛**：现场抽签展示，评委打分

## 七、各层级备赛建议

### 校赛备赛（基础起点）

| 重点 | 建议 |
|------|------|
| 选题 | 选择自己最熟悉、最有把握的内容 |
| 教学设计 | 确保格式规范，要素完整 |
| PPT | 简洁清晰，不过度设计 |
| 课堂展示 | 重点练好时间控制和语言规范 |
| 教学反思 | 掌握三维度框架，练习快速撰写 |

### 省赛备赛（突出特色）

| 重点 | 建议 |
|------|------|
| 选题 | "获得感+彰优势+要素全"，体现学科特色 |
| 教学设计 | 5个学时需逻辑连贯，体现课程整体设计 |
| PPT | 20页左右，结构化+精悍性+美观度 |
| 课堂展示 | 突出教学特色，有1-2个创新点 |
| 教学反思 | 有深度，有具体不足和改进方向 |
| 关键 | 邀请专家指导，反复模拟演练 |

### 国赛备赛（最高标准）

| 重点 | 建议 |
|------|------|
| 选题 | 16个节段覆盖整本教材，每个节段都要精 |
| 教学设计 | 16份教学设计，9万余字级别 |
| PPT | 16套PPT，每套20页左右 |
| 讲稿 | 6万余字讲稿，每个节段熟练 |
| 磨课 | 录制近百个磨课视频 |
| 课堂展示 | 现场抽签，每个节段都要准备好 |
| 教学反思 | 45分钟无资料现场撰写，需内化框架 |
| 关键 | 省级专家2轮集中打磨+校级全程支持+教研室集体备课 |

### 国赛备赛参考数据
> 燕山大学纪淳老师（国赛二等奖）：
> - 9万余字教学设计
> - 6万余字讲稿
> - 录制近百个磨课视频
> - 16个教学节段覆盖整本教材

## 八、时间规划建议

| 层级 | 建议备赛时间 | 关键节点 |
|------|-------------|----------|
| 校赛 | 4-6周 | 第2周定选题，第4周完成初稿，第6周模拟演练 |
| 省赛 | 8-10周 | 第4周完成5个教学设计，第6周专家指导，第10周冲刺 |
| 国赛 | 12-16周 | 第8周完成16个教学设计，第10周专家2轮打磨，第14周全流程模拟 |
