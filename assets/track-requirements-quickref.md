## 文科组速查

### 学科范围
哲学/经济学/法学/教育学/文学/历史学/管理学/艺术学

### 必须做到
- [ ] 文本深度解读，多视角分析
- [ ] 文化比较，经典与新锐结合
- [ ] 讨论式/案例式教学方法
- [ ] 涵养人文素养与家国情怀
- [ ] 精准表达，知识点逻辑衔接精雕细琢

### 避免事项
- ❌ 理论空泛缺乏实例
- ❌ 单向灌输无互动
- ❌ 思政融入生硬
- ❌ 内容过于浅层

### 获奖密码
> "比赛是日常教学的'极致化呈现'，需在知识点逻辑衔接、讲授思路设计乃至教态仪表上精雕细琢"

---

## 理科组速查

### 学科范围
数学/物理学/化学/天文学/地理学/大气科学/海洋科学/地球物理学/地质学/生物学/系统科学/生态学/统计学等

### 必须做到
- [ ] 概念建构过程清晰
- [ ] 推理链条完整严密
- [ ] 实验验证/数学语言表达
- [ ] 科学精神、求真务实
- [ ] 跨学科认知链条

### 避免事项
- ❌ 公式推导枯燥
- ❌ 概念抽象缺乏直观
- ❌ 前沿联系不足
- ❌ 纯讲授无互动

### 获奖密码
> "以'目标-评估-活动'一致性原则重构教学设计，将课堂打磨为'钩子-知识点-高潮-留白'节奏，打破学科壁垒"

---

## 工科组速查

### 学科范围
工学/农学

### 必须做到
- [ ] 工程案例驱动
- [ ] 体现"两性一度"（高阶性/创新性/挑战度）
- [ ] 前沿科技与产业需求
- [ ] 工匠精神、工程伦理
- [ ] 理论与实践紧密结合

### 避免事项
- ❌ 理论与实践脱节
- ❌ 案例过时
- ❌ 缺乏互动设计
- ❌ 工程伦理缺失

### 获奖密码
> "选题遵循'获得感、彰优势、要素全'原则，采用BOPPPS模型，构建'问题导入——知识建构——实践应用——价值升华'闭环"

---

## 医科组速查

### 学科范围
基础医学/临床医学/口腔医学/公共卫生/中医学/中西医结合/药学/中药学/护理学/法医学等

### 必须做到
- [ ] 疾病-机制-诊疗认知链条
- [ ] 基础医学与临床医学深度融合
- [ ] CBL/PBL/模拟教学方法
- [ ] 严谨行医、以患者为中心
- [ ] 从"知识传递"转向"能力孵化"

### 避免事项
- ❌ 基础临床割裂
- ❌ 案例不典型或过时
- ❌ 伦理讨论缺失
- ❌ 缺乏临床思维引导

### 获奖密码
> "打破学科壁垒，让课堂成为学生临床思维与创新能力的孵化器"

### 联动Skill
- PBL医学教学工具箱：多幕式案例、基础-临床桥接
- ISW医学院教师培训：BOPPPS教案、微格教学

## 思想政治课专项组速查

### 学科范围
马克思主义基本原理/毛泽东思想和中国特色社会主义理论体系概论/中国近现代史纲要/思想道德与法治/形势与政策等

### 必须做到
- [ ] 把道理"讲深讲透讲活"
- [ ] 问题链闭环设计
- [ ] 理论联系实际
- [ ] 青年叙事转化
- [ ] "兴趣驱动+深度思考+创意思维"三维体系

### 避免事项
- ❌ 抽象说教
- ❌ 学生需求关注不足
- ❌ 理论与实际脱节
- ❌ 课堂氛围沉闷

### 获奖密码
> "将'大道理'转化为'青年叙事'，将天边榜样转化为身边故事，将'高大上的理想'和'人间烟火气'紧密融合"

## 跨组别选择建议

### 学科边界判断
| 课程类型 | 推荐组别 | 备注 |
|----------|----------|------|
| 医学中的生物学基础 | 医科 | 非理科 |
| 农学类课程 | 工科 | 含农学 |
| 艺术学课程 | 文科 | |
| 管理学课程 | 文科 | |
| 心理学课程 | 文科（教育学） | |
| 统计学课程 | 理科 | |

### 选组策略
1. 优先选择能充分发挥个人优势的组别
2. 考虑组别竞争激烈程度
3. 参考往届本校该组别获奖情况
4. 选择有专家指导资源的组别
