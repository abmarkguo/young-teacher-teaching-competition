# 竞赛PPT设计与板书配合指南

本文档提供全国高校青年教师教学竞赛PPT课件设计和板书配合的专业指南。

---

## 一、PPT设计四大要素

### 核心原则
> **结构化、精悍性、美观度、富媒体**

### 1.1 结构化

| 要求 | 标准 |
|------|------|
| 页数控制 | 20分钟课堂约20页（±3页） |
| 每页核心 | 每页一个核心观点，不堆砌 |
| 逻辑层次 | 层次清晰，与教学设计一致 |
| 节奏匹配 | 翻页节奏与教学节奏匹配 |

**页面结构建议**：
```
封面页（1页）：课程名称+章节
目录页（1页）：本节内容框架
导入页（1-2页）：问题/案例/悬念
知识页（8-10页）：核心知识建构
应用页（3-5页）：实践/案例/探究
总结页（1-2页）：知识梳理+思政升华
延伸页（1页）：课后任务/思考题
```

### 1.2 精悍性

| 要求 | 标准 |
|------|------|
| 文字精炼 | 每页不超过6行文字，每行不超过15字 |
| 图表优先 | 以图表、动画替代大段文字 |
| 关键突出 | 核心概念加粗/变色/放大 |
| 留白充足 | 页面留白≥30%，不拥挤 |

**文字处理原则**：
- 讲稿文字≠PPT文字
- PPT只放关键词、核心句
- 详细内容由教师口头展开
- 用图表表达关系和过程

### 1.3 美观度

| 要求 | 标准 |
|------|------|
| 配色 | 专业统一，主色不超过3种，符合学科气质 |
| 字体 | 标题≥28pt，正文≥18pt，中文字体统一 |
| 排版 | 对齐整齐，间距统一，不杂乱 |
| 图片 | 高清无水印，与内容相关，风格统一 |

**配色建议**：

| 学科组 | 推荐主色 | 气质 |
|--------|----------|------|
| 文科 | 暖色调（棕/金/米） | 人文典雅 |
| 理科 | 冷色调（蓝/灰/白） | 理性严谨 |
| 工科 | 科技色（深蓝/银/橙） | 科技现代 |
| 医科 | 生命色（青/白/绿） | 专业洁净 |
| 思政 | 庄重色（红/金/白） | 庄重大气 |

### 1.4 富媒体

| 类型 | 使用建议 |
|------|----------|
| 动画 | 页面切换动画简洁，内容动画服务于教学 |
| 视频 | 短视频（30秒-1分钟），嵌入而非链接 |
| 仿真 | 适当使用仿真演示，动态展示过程 |
| 交互 | 适当使用触发器、超链接等交互元素 |

**注意**：富媒体是加分项，但不能喧宾夺主，所有媒体必须服务于教学目标。

## 二、PPT内容设计规范

### 2.1 封面页
- 课程名称（大字号居中）
- 章节信息（副标题）
- 简洁大方，不花哨
- 无个人信息（竞赛匿名要求）

### 2.2 导入页
- 用问题/案例/数据/图片/视频开场
- 视觉冲击力强
- 激发好奇心
- 与教学目标建立连接

### 2.3 知识页
- 核心概念用图表/思维导图呈现
- 公式/推导过程用动画逐步展示
- 复杂过程用流程图/示意图
- 对比内容用表格/对比图

### 2.4 互动页
- 互动问题清晰展示
- 讨论话题简洁明确
- 可设计投票/选择等互动元素
- 预留思考空间

### 2.5 案例页
- 案例背景简洁交代
- 关键信息突出显示
- 分析过程可视化
- 结论与知识点呼应

### 2.6 思政融入页
- 思政元素视觉化呈现
- 自然融入，不单独设立"思政环节"
- 用图片/数据/故事引出价值观
- 不堆砌口号

### 2.7 总结页
- 知识结构图/思维导图
- 核心要点回顾
- 思政升华自然点睛
- 悬念/思考题延伸

## 三、板书设计规范

### 3.1 板书与PPT的分工

| 内容 | PPT | 板书 |
|------|-----|------|
| 结构框架 | ✅ 主力 | 辅助 |
| 图表图片 | ✅ 主力 | — |
| 推导过程 | 辅助 | ✅ 主力 |
| 关键词概念 | 辅助 | ✅ 主力 |
| 动态过程 | ✅ 动画 | — |
| 互动记录 | — | ✅ 记录学生回答 |

### 3.2 板书布局

```
┌─────────────────────────────────────────┐
│  [标题区]                                 │
│                                          │
│  [主要内容区]          [辅助区/学生回答]    │
│  推导过程/关键词        互动记录/补充说明    │
│                                          │
│  [总结区]                                 │
│  核心结论                                 │
└─────────────────────────────────────────┘
```

### 3.3 板书要求
- 简洁工整美观
- 大小适当（最后一排可见）
- 结构合理，有层次
- 与PPT配合，不重复
- 书写及时，边讲边写

## 四、AI赋能教学在PPT中的呈现

### 4.1 展示方式
- 截图展示AI工具界面
- 录屏嵌入展示AI应用过程
- 现场演示（如果条件允许）
- 数据可视化展示AI分析结果

### 4.2 应用场景展示

| 场景 | PPT展示方式 |
|------|------------|
| 个性化学习路径 | AI推荐路径截图+说明 |
| 学情分析 | 学习数据仪表盘截图 |
| 智能评价 | AI批改结果对比 |
| 虚拟实验 | 仿真操作截图/视频 |
| 知识图谱 | 可视化知识网络图 |

### 4.3 注意事项
- AI应用要真实可信，不虚构
- 要说明AI如何服务于教学目标
- 展示AI应用的教学效果
- 体现"教师为主导，AI为辅助"

## 五、PPT评审清单

### 结构化（25分）
- [ ] 页数控制在20页左右（±3页）
- [ ] 每页一个核心观点
- [ ] 逻辑层次清晰
- [ ] 与教学设计完全一致
- [ ] 翻页节奏匹配教学节奏

### 精悍性（25分）
- [ ] 每页文字不超过6行
- [ ] 以图表替代大段文字
- [ ] 关键信息突出
- [ ] 留白充足（≥30%）
- [ ] 无照念PPT的情况

### 美观度（25分）
- [ ] 配色专业统一（主色≤3种）
- [ ] 字体大小达标（标题≥28pt，正文≥18pt）
- [ ] 排版整齐统一
- [ ] 图片高清相关
- [ ] 整体风格协调

### 富媒体（10分）
- [ ] 动画恰当有效
- [ ] 视频/仿真适度
- [ ] 交互元素合理
- [ ] 不喧宾夺主

### 内容配合（15分）
- [ ] 思政元素自然融入
- [ ] AI赋能教学展示
- [ ] 与板书互补不重复
- [ ] 前沿内容呈现
- [ ] 案例真实典型

## 六、常见PPT问题

| 问题类型 | 具体表现 | 改进方案 |
|----------|----------|----------|
| 文字过多 | 大段文字照念 | 提炼关键词，用图表替代 |
| 配色混乱 | 颜色过多不统一 | 确定主色方案，统一应用 |
| 字体过小 | 后排看不清 | 标题≥28pt，正文≥18pt |
| 动画过度 | 花哨动画分散注意 | 简洁动画，服务于教学 |
| 结构混乱 | 逻辑不清 | 梳理教学逻辑，重新排版 |
| 图片模糊 | 低清图片拉伸 | 使用高清图片或重制 |
| 内容冗余 | 信息过载 | 精简内容，突出重点 |
| 缺乏互动 | 纯展示无互动 | 增加互动页面设计 |
| 无AI元素 | 未展示AI应用 | 至少1个AI应用场景 |
| 思政突兀 | 突然出现思政页 | 自然融入各页面 |
