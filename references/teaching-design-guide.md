# 教学设计撰写专业指南

本文档提供全国高校青年教师教学竞赛教学设计（20%权重）的专业撰写指南。

---

## 一、教学设计基本框架

### 必需要素

```
1. 课程名称（注明对应的参赛课程大纲章节）
2. 教学指导思想
3. 内容分析
4. 学情分析
5. 教学目标（知识/能力/思政育人三维）
6. 教学重点与难点
7. 教学方法与工具
8. 教学过程设计
9. 教学评价
10. 预习任务与课后作业
11. 课程资源
```

## 二、各要素撰写规范

### 2.1 课程名称
- 注明对应的参赛课程大纲章节
- 示例：「生物化学——第3章 酶学（第2学时：酶促反应动力学）」

### 2.2 教学指导思想
- 体现"以学生为中心"理念
- 体现立德树人根本任务
- 说明所采用的教学模型（BOPPPS/五星教学/翻转课堂等）
- 字数：200-300字

**范例**：
> 本节课秉持"以学生为中心"的教学理念，围绕立德树人根本任务，采用BOPPPS教学模型，通过"问题导入——知识建构——实践应用——价值升华"的闭环逻辑，引导学生在探究中建构知识，在应用中提升能力，在反思中涵养素养。结合AI辅助教学工具，实现个性化学习路径推荐和实时学情分析。

### 2.3 内容分析
- 本节内容在课程中的地位和作用
- 与前后知识的衔接关系
- 内容的学术价值和实践意义
- 前沿发展趋势
- 字数：300-400字

### 2.4 学情分析
- 学生已有知识基础
- 学生认知特点和学习习惯
- 可能的学习困难和误区
- 学生的专业背景和需求
- 字数：300-400字

**关键要点**：学情分析要具体到本节课，不能泛泛而谈"学生已有一定基础"。

### 2.5 教学目标

#### 三维目标体系

**知识目标**：
- 掌握……（核心概念/原理/方法）
- 理解……（机制/关系/过程）
- 了解……（前沿/应用/发展）

**能力目标**：
- 能够运用……解决……问题
- 具备……能力（分析/推理/设计/评价）
- 培养……思维（批判性/创新性/临床）

**思政育人目标**：
- 涵养……素养（科学精神/人文情怀/职业伦理）
- 树立……意识（责任意识/创新意识/服务意识）
- 增强……自信（文化自信/道路自信/专业自信）

**注意**：思政目标不用写得宏大空洞，贴合本节课教学内容和学生专业特点即可。

### 2.6 教学重点与难点

**重点**（2-3个）：本节课最核心、最关键的知识点或能力
**难点**（1-2个）：学生最不容易理解或掌握的内容

每个重难点需配套：
- 突破策略
- 具体方法
- 时间安排

### 2.7 教学方法与工具

| 类别 | 具体方法/工具 |
|------|-------------|
| 教学方法 | 讲授法/启发式/探究式/案例式/项目式/翻转课堂 |
| 互动方式 | 提问/讨论/小组合作/角色扮演/辩论 |
| 教学工具 | PPT/板书/模型/视频/仿真软件/在线平台 |
| AI赋能工具 | 智能推荐/学习分析/自动评估/虚拟实验 |

### 2.8 教学过程设计

#### 推荐教学模型

**BOPPPS模型**：
- B (Bridge-in) 桥接导入：吸引注意，建立连接
- O (Objective) 学习目标：明确目标，引导方向
- P (Pre-assessment) 前测：了解基础，调整教学
- P (Participatory Learning) 参与式学习：核心环节，互动探究
- P (Post-assessment) 后测：检验效果，反馈调整
- S (Summary) 总结：梳理要点，拓展延伸

**五星教学模型**：
1. 聚焦问题
2. 激活旧知
3. 示证新知
4. 尝试应用
5. 融会贯通

**20分钟课堂节奏设计**：
- 钩子（2-3分钟）：问题导入，吸引注意
- 知识点（8-10分钟）：核心知识建构
- 高潮（5-6分钟）：实践应用或深度探究
- 留白（2-3分钟）：总结反思，悬念延伸

#### 过程设计格式

```
环节一：导入（X分钟）
- 教师活动：……
- 学生活动：……
- 设计意图：……
- 课程思政融入：……（如适用）

环节二：新知建构（X分钟）
- 教师活动：……
- 学生活动：……
- 设计意图：……
- 重难点突破：……

……（依此类推）

环节N：总结与延伸（X分钟）
- 知识梳理：……
- 思政升华：……
- 课后延伸：……
```

### 2.9 教学评价
- 过程性评价：课堂提问/互动参与/小组表现
- 结果性评价：课后作业/测验/项目
- 评价标准：具体可操作的评分量规

### 2.10 预习任务与课后作业
- 预习任务：具体、可检查
- 课后作业：分层设计（基础/提升/拓展）
- 作业形式多样

### 2.11 课程资源
- 教材及参考书
- 在线资源（MOOC/网站/数据库）
- AI工具推荐
- 拓展阅读文献

## 三、课程思政融入策略

### 核心原则
> **"像盐溶于水"——不着痕迹却浸润人心**

### 融入方法

1. **知识点对应法**：每个重点知识点对应一项贴合的思政内涵
2. **全流程融入法**：均匀融入课堂导入、探究、小结、作业四个环节
3. **案例渗透法**：用真实案例自然引出价值观讨论
4. **问题引导法**：通过问题设计引导学生自主思考价值议题

### 学科特色思政要点

| 学科组 | 思政切入点 |
|--------|-----------|
| 文科 | 文化自信、人文情怀、社会责任 |
| 理科 | 科学精神、求真务实、服务国家战略 |
| 工科 | 工匠精神、工程伦理、产业报国 |
| 医科 | 医者仁心、敬畏生命、以患者为中心 |
| 思政 | 理论深度、价值认同、使命担当 |

### 常见失分点
- 思政内容突兀插入，与专业知识脱节
- 价值观说教化，口号堆砌
- 仅在一个环节融入，分布不均
- 思政目标空洞宏大，不贴合课堂实际

## 四、AI赋能教学设计

### 评分要求
> AI赋能教学应用已纳入课堂教学评分维度（教学手段4分中占2分）

### AI应用场景

| 场景 | 应用方式 |
|------|----------|
| 个性化学习 | AI推荐学习路径、智能题库 |
| 学情分析 | 学习行为分析、掌握度诊断 |
| 智能评价 | 自动批改、即时反馈 |
| 虚拟实验 | AI仿真、虚拟操作 |
| 知识图谱 | 智能关联、可视化呈现 |
| 课堂互动 | AI问答助手、智能弹幕 |

### 设计要点
- AI应用要服务于教学目标，不能为技术而技术
- 要体现"教师为主导，学生为中心"的理念
- AI应用效果要有可检验的方式

## 五、格式规范

### 字体要求
- 一级标题：3号黑体加粗
- 二级标题：4号黑体加粗
- 三级标题：小4号黑体加粗
- 正文内容：小4号宋体，1.5倍行距

### 封面格式
- 第一行：竞赛材料名称（页面左上角，二号，宋体）
- 第二行：课程名称（页面三分之一处居中，初号，宋体）
- 第三行：学院及姓名（页面二分之一处居中，一号，宋体）

### 其他要求
- 格式为PDF文档通用格式
- 教学材料不得出现姓名、学院(系)等任何个人相关信息
- 教学大纲装订在教学设计内容之前
